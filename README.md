# WarPlane

## ✈️ Sobre o projeto

Este projeto é um jogo desenvolvido em **Python** utilizando a biblioteca **Pygame**, no qual o jogador controla um avião em voo contínuo.

## 🎮 Jogabilidade

O objetivo é sobreviver o máximo possível, desviando de obstáculos e enfrentando inimigos que surgem ao longo do caminho.

O jogo envolve:
- Movimentação do avião
- Detecção de colisões
- Geração dinâmica de inimigos e obstáculos

## 🚀 Objetivo

Proporcionar uma experiência simples, porém envolvente, focada em **reflexos** e **tomada rápida de decisões**.
